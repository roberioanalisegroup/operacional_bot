# 🚀 Como Usar a Aplicação Web

## Passo a Passo Rápido

### 1️⃣ Abrir a Aplicação

**Opção Simples:**
- Clique duas vezes no arquivo `index.html`
- Ele abrirá no seu navegador padrão

**Opção com Servidor (Recomendado):**
```bash
# No terminal, na pasta do projeto:
python -m http.server 8000
```
Depois acesse: http://localhost:8000

### 2️⃣ Selecionar Arquivo SPED

- Clique na área "Clique para selecionar ou arraste o arquivo SPED"
- OU arraste o arquivo .txt para a área
- O nome do arquivo aparecerá quando selecionado

### 3️⃣ Processar

- Clique no botão "Processar Arquivo"
- Aguarde o processamento (alguns segundos)
- Os dados aparecerão automaticamente

### 4️⃣ Visualizar Resultados

Você verá:
- **Estatísticas**: Total de registros, valor total, códigos encontrados
- **Tabela Completa**: Todos os códigos com valores, COFINS e PIS calculados

### 5️⃣ Exportar

**Criar Planilha Modelo:**
- Clique em "Criar Planilha Modelo"
- Uma planilha Excel será baixada com todos os códigos

**Exportar Planilha Preenchida:**
- Clique em "Exportar Planilha Preenchida"
- Uma planilha Excel será baixada com os dados processados

## ✅ Pronto!

A aplicação funciona completamente no navegador, sem necessidade de instalar nada!

## 🧪 Teste Rápido

Use o arquivo `exemplo_sped.txt` incluído para testar a aplicação.

## 💡 Dicas

- Funciona offline após carregar a página
- Seus dados nunca saem do seu computador
- Funciona em qualquer sistema operacional
- Não precisa de Python ou outras instalações
